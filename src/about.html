About This Project

This is a proof-of-concept for hosting a fully private, media-rich website on a public static host like GitHub Pages.

Technology Stack

  • DuckDB-Wasm — In-browser analytical database
  • AES Encryption — Block-level encryption of the database file
  • HTTP Range Requests — Fetch only needed data blocks
  • Web Crypto API — Secure key handling in the browser
