Welcome to the Private Site

This entire website is served from an encrypted DuckDB database.

No plaintext assets exist on the server — everything is decrypted in your browser.

How it works

  1. All site files are packed into an encrypted DuckDB database at build time
  2. The browser fetches only the encrypted blocks it needs via HTTP range requests
  3. DuckDB-Wasm decrypts and serves content entirely client-side
